dlib
====

INSTALLATION

1. Install pip
Download https://bootstrap.pypa.io/get-pip.py
Run: sudo python get-pip.py

2. Install django
Run: sudo pip install django==1.5.8

3. Update DB
Run: python manage.py syncdb

4. Run server
Run: python manage.py runserver

5. Open site
URL: http://127.0.0.1:8000/
